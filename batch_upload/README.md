# Carga en Lote (Batch Upload) 📂🚀

Este script te permite subir todas tus fotos de golpe al servidor. Ya no importa cómo las nombres, el sistema las registrará y luego tú les pones los detalles en el Admin.

### 📝 Instrucciones:

1.  **Copia tus fotos**: Pega todas las imágenes que quieras subir aquí mismo (en esta carpeta `batch_upload/`).
2.  **No importa el nombre**: Puedes llamarlas como quieras (ej: `foto1.jpg`, `jersey_nuevo.png`). El sistema usará ese nombre como inicial.
3.  **Ejecuta el Script**: Abre una terminal en tu proyecto y escribe:
    ```bash
    node scripts/batch-upload.js
    ```
4.  **Edita en el Admin**: Una vez que termine el script, ve a tu panel de Administrador. Verás los nuevos productos allí con la categoría "Adulto" y talla "?". Haz clic en el botón de **"E" (Editar)** para ponerles el nombre final, el precio y la talla correcta.

¡Es así de simple! ✨
